#analysis 

(of a set $A$) the set of points $a \in R$ such that
- $\forall \epsilon > 0, V_\epsilon (a) \cap Q \neq \emptyset$ and
- $\forall \epsilon > 0$, $V_\epsilon (a) \cap Q^C \neq \emptyset$ 


This set is denoted $\partial A$.
